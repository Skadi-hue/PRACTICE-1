# Telegram Bot «Рандомайзер»

Бот для генерации случайных чисел и случайного выбора элементов из списка. Работает в Telegram.

## Возможности

- Генерация случайного целого или дробного числа в заданном диапазоне.
- Поддержка как положительных, так и отрицательных чисел.
- Выбор случайной строки из многострочного сообщения или из списка, разделённого запятыми/точкой с запятой.
- Кнопка «старт» для повторного вывода справки.

## Команды

- `/start` – выводит приветствие и описание возможностей.

## Форматы ввода

### Одно число

- Введите одно число – бот вернёт случайное число от 0 до указанного (если число положительное) или от него до 0 (если отрицательное).
- Поддерживаются как целые, так и дробные числа (разделитель – точка или запятая).

**Примеры:**  
`10` → `7`  
`-5.5` → `-2.3`  
`3,14` → `1.57`

### Два числа

- Введите два числа через пробел, запятую, точку с запятой или двоеточие.
- Бот вернёт случайное число между ними.
- Если оба числа целые – результат будет целым, иначе – дробным.

**Примеры:**  
`1, 100` → `42`  
`-10; 10` → `3`  
`1.5 : 5.5` → `3.78`

### Список строк (через перенос строки)

- Отправьте несколько строк (каждая с новой строки) – бот выберет одну случайную строку.

**Пример:**  
`яблоко`     
`груша`   
`апельсин`   

→ `груша`
