VIDEOTEXT

VIDEOTEXT — Функциональность в составе веб-проекта Syllabica, обеспечивающая интеграцию модуля анимации в основную кодовую базу (Flask) и организацию асинхронной очереди для выполнения ресурсоёмких задач рендеринга.

ЦЕЛЬ

Создание анимированных видеороликов на основе текстовых сценариев, работа с генетическими архивами, инструменты для филологов (импорт из DOCX, редактирование сценариев, рендеринг видео, наглядное представление данных)

Журнал

Команда проекта

🔹 2 марта 2026 – Завершена разработка портативного компилятора C++ в JSON с графическим интерфейсом на C#

Команда VIDEOTEXT полностью переработала устаревший движок, привязанный к Qt. Проведён рефакторинг на стандартный C++17, создана консольная утилита s_compiler.exe и легковесная обёртка на C# Windows Forms. Реализован асинхронный запуск, автоопределение кодировки (UTF-8 / Windows-1251), Drag‑and‑Drop, пакетная обработка, встроенный JSON‑вьюер и история операций.
✅ Итог: выпущена Portable-версия приложения, работающая на любом Windows-ПК без установки дополнительных библиотек.

О проекте

Syllabica (образовательный веб-проект)

Специализация команды

VIDEOTEXT – реализация функциональности видеотекста

Цель: создание анимированных видеороликов на основе текстовых сценариев, работа с генетическими архивами, инструменты для филологов (импорт из DOCX, редактирование сценариев, рендеринг видео, наглядное представление данных)

Технический стек

Backend

  • Python 3 (Flask, FastAPI – только прототип)
  • C++17 (консольное ядро s_compiler.exe)
  • C# .NET Windows Forms (GUI для компилятора)
  • Асинхронность: queue.Queue (Producer-Consumer), async/await, Task.Run
  • Обработка документов: python-docx, XML-парсинг (w:shd для цвета фона)
  • Видеорендеринг: Pygame, FFmpeg
  • Система сборки: Release-конфигурация для C++ и C#

Frontend

  • HTML5 (семантическая вёрстка)
  • CSS3 (CSS Grid, Flexbox, медиазапросы, переменные)
  • Bootstrap (интерфейс редактора)
  • Ace Editor (редактор JSON с подсветкой синтаксиса, тема «monokai»)
  • JavaScript (fetch) – асинхронные запросы к API

Инструменты проектирования и управления

  • Figma – дизайн макетов
  • Git – система контроля версий
  • Git-репозиторий: https://git.syllabica.com/syllabica-dev.git

Библиотеки и зависимости (Python)

  • Flask, FastAPI (только legacy)
  • python-docx
  • pygame
  • ffmpeg (вызов через subprocess)
  • Стандартные модули: queue, subprocess, json, os, pathlib
